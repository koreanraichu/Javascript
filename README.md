# Welcome to my repository!
보통 자잘한 토이프로젝트가 주로 올라갑니다. 언제 올라가냐고요? 그건 소재가 생겨야 압니다. 소재라는 게 매일 팟! 하고 생기는 게 아니라... 
블로그건 사이트건 참고해서 하는 경우에는 주석에 넣거나 커밋하면서 참고한 사이트 URL을 올리고 있습니다. 

각 폴더 자체가 프로젝트고, 그 한 폴더에 HTML, CSS, JS파일이 한번에 다 들어있습니다. 
그 파일을 어디다가 다운로드하건 상관은 없는데, 세 파일이 전부 같은 위치에 있어야 합니다. 다른 위치에 있으면 CSS고 JS고 안 먹힙니다.
